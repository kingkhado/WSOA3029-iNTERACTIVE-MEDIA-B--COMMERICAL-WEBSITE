THE DESIGN STORY

HHD

Welcome to the Design Section of the HHD Homeless Shelter Organization website. Here, you will find wireframes and style guides that reflect the thoughtful decisions made in both the user interface (UI) and user experience (UX) design. Each design choice aims to create an intuitive, accessible, and visually engaging platform, supporting our mission to raise awareness and provide resources for the homeless. This section showcases the rationale behind the layouts, color schemes, typography, and interactive elements that make the HHD site both functional and user-friendly.

Wireframe 1: Homepage Layout

This layout presents the main navigation, featured content, and call-to-action buttons, designed to guide users through the website's core sections effectively.

Wireframe 2: About Us Layout

This section introduces the organization’s mission clearly and builds trust with visitors. A house-and-heart icon reinforces the mission visually, adding an inviting touch. Key statistics on poverty and homelessness are displayed in structured cards, helping users grasp the issue's scope at a glance while maintaining a balanced, accessible layout.

Wireframe 3: Join The Movement Layout

The "Join the Movement" page uses icons paired with concise text to clearly convey specific goals and actions. The illustrations create a dynamic, engaging visual flow, allowing visitors to quickly grasp mission areas without heavy reading.

Wireframe 4: How to Help Layout

The "How to Help" section presents multiple donation options with clear pricing to simplify decision-making and accommodate various contributions. Each option, like "Provide Clothing" or "Provide Shelter," is paired with an icon and label, creating a uniform, easy-to-scan layout that keeps the page visually balanced.

Wireframe 5: Why Your Effort Matters Layout

The "Why Your Effort Matters" section uses a heartwarming image and bullet points to highlight the impact of donations, making an emotional connection with users. Checkmarks beside each point improve readability, guiding users through the benefits of contributing and reinforcing the importance of their support for the cause.

Wireframe 6: Contact Us Layout

The contact page wireframe focuses on providing an intuitive form for user inquiries, with easy-to-find contact details and a map integration for location reference.

Wireframe 7: footer Scroll Up Layout

The footer’s scroll-up button, a black box with a white arrow, appears upon reaching the bottom of any page, offering a quick return to the top. Its subtle spring-up animation catches attention without distraction, enhancing usability with a smooth, responsive navigation option.

Wireframe 8: Testimonial Layout

The **Testimonial** section uses a powerful before-and-after image to showcase the transformative impact of support, aligning with the mission to aid those in need. A “SEE MORE” call to action invites users to explore personal journeys, building empathy and connection. Navigation arrows allow seamless browsing between testimonials, while the **"SEE ALL"** button lets users return to the main testimonials page effortlessly, enhancing usability and encouraging further exploration of impactful stories.

Wireframe 9: Donate Layout

The **Donate Page** emphasizes the theme “Support Our Cause” with bold, vertical “D O N A T E” text for a memorable visual effect. Each donation item—like Free Books, Clothing, or Shelter Fare—appears in a card with an image, donation amount (from R0 to R100), and an “Add To Cart” button, making options clear and easy to select. A cart button provides a quick summary of selected items and options to checkout or close, streamlining the donation process for users.

Wireframe 10: Node Edge VisualLayout

The Node-Edge Visual Wireframe highlights Bible verses on giving to the poor, with each verse displayed as a node organized by chapter within a toolkit. Lines connect related verses, forming a visual network that emphasizes themes of generosity, kindness, and support for those in need. Color-coded nodes differentiate chapters, enhancing clarity and helping users follow connections as they explore related scriptures in an interactive, cohesive format.

Wireframe 11: Bar Graph Layout

The **Homeless Population Bar Chart Page** for South Africa features an interactive bar graph displaying population data across provinces, with a focus on homelessness. Customizable buttons enhance user experience, allowing toggling between general and homeless population data. Sorting options let users organize the data in ascending or descending order by population size, while a reset button restores the chart to its original view. This setup provides clear, interactive insights into homelessness and population distribution across provinces.

Wireframe 12: Design Layout

The Design section introduces the design process with the title and an overview that explains the website's UI/UX foundation, fostering transparency and trust. The wireframe highlights the layout and design rationale, providing users with insight into the creation of a user-friendly interface. Simple navigation through "PREV" and "NEXT" buttons ensures a smooth, continuous transition between design phases or sections, enhancing the user experience.

Wireframe 13: Theory Layout

**The Theory** section offers a detailed explanation of the design’s analytical foundation, with the heading and description guiding users through the purpose and decisions behind each visualization. Dot navigation enhances interactivity and organization, allowing users to easily explore different essays. The "Read More," "Preview," and "Download" buttons in the essay preview section provide flexible engagement options, catering to varying levels of interest in the content.

Style Guide: Colour Scheme

#2C4A68 (Deep Blue) – This strong shade of blue is used for accent headings and buttons, conveying trust, stability, and compassion. Blue is often associated with calmness and reliability, making it ideal for a website that seeks to inspire trust and motivate donations for a charitable cause. The color creates visual interest while maintaining the professional tone needed for a non-profit organization. #17798B0 (Bright Blue) – This lighter blue tone is used for highlighting key actions like “Donate Now.” It draws attention without overwhelming the user, encouraging interaction and engagement. Blue tones are commonly used in charity websites as they evoke feelings of care and empathy, aligning with the mission of helping those in need. #ADAFB9 (Light Gray) – This shade is applied to background sections to soften the high contrast between dark and light elements. It ensures readability without causing eye strain, creating a calm and professional look that allows users to focus on the information presented. >#FAFBFB (White) – White is used for text and button backgrounds. This neutral color provides high contrast against dark backgrounds, enhancing readability. It contributes to a clean and minimalistic look, allowing the primary colors to stand out, which is essential for user navigation. #000000 (Black) – The deep black color is used as the primary background color, creating a sleek, modern, and professional aesthetic. The black background enhances the readability of white and blue text, making key messages and calls to action stand out. The dark theme reflects the serious nature of the cause, resonating with the user on an emotional level.

Style Guide: Font Choice

Montserrat Sans-Serif – Montserrat has been chosen for its modern and clean style, which aligns well with the serious and professional tone of the website. Montserrat is easy to read, even in smaller sizes, making it an excellent choice for both headings and body text. The typeface’s clarity ensures effective communication, especially in conveying essential information about donations and support options. Font Weight and Hierarchy: Bold for headings, such as “HELP THE HOMELESS DONATE NOW” – Bold weights emphasize urgency and importance, guiding the user’s attention to key messages and calls to action. Regular for body text – Regular weight is used for descriptive text and additional information, maintaining readability and a clean appearance

Style Guide: Brand Icon

The website centers around supporting charitable causes, with an emphasis on building a community of care and trust. To reflect this mission, I named the platform “Helping Hands Donate”, emphasizing the focus on compassion and aid. The logo visually conveys this purpose: it combines a simple outline of a house with an open hand at its center. This design represents both the comfort and security of a home and the welcoming, supportive nature of an outstretched hand. > The hand within the house symbol is a powerful and memorable brand identity that aligns with our commitment to helping others. The logo suggests that charity begins at home and that even small, individual acts of kindness can create a safe and nurturing environment. The clean lines and thoughtful symbolism make the logo approachable and trustworthy, reflecting the values at the heart of this non-profit platform. The color palette, a mix of deep and bright blues, light gray, and white, complements this theme. These colors evoke a sense of calm, empathy, and security, making the platform inviting while maintaining a professional, credible aesthetic. To further establish trust and enhance credibility, icons for recognized, secure payment options are prominently displayed. This reassures users that their donations are handled with care and security, building confidence in the website’s transaction process.

Style Guide: Header Navigation

The navigation bar on HHD is designed for a smooth and intuitive browsing experience. The active page is highlighted in a distinct blue, allowing users to instantly recognize their current location on the site. Each navigation item is interactive; hovering over an option reveals a pointer cursor, inviting users to explore further. The layout ensures ample spacing between links, providing both readability and a refined aesthetic, making it easy for visitors to navigate the content effortlessly

Style Guide: Final Remarks

Dark Background with High-Contrast Elements – The decision to use a dark background with contrasting white and blue text elements emphasizes readability and gives the site a modern, impactful look. It reflects the seriousness of the cause and encourages a focused experience without distractions. Rounded Iconography and Buttons: – The rounded shapes in icons and buttons create a sense of approachability, inviting users to engage without feeling overwhelmed. This softens the serious tone of the website, making it feel supportive and encouraging rather than overly stern. Strengths: Emotionally Engaging – The combination of dark tones, powerful images, and compelling calls to action creates a strong emotional appeal, which is crucial for encouraging donations. Clear Information Flow: – The hierarchical layout ensures users can quickly find information and navigate the site with ease. Professional and Trustworthy: – The use of blue and black tones, alongside clear data and testimonials, enhances the site's credibility and encourages trust in the organization. Weaknesses: Limited Color Palette – While the dark theme aligns well with the mission, it may feel heavy or somber to some users. Adding a few more warm tones might help balance the mood. Minimal Animation/Interactivity: – Adding subtle animations, such as hover effects on buttons, could enhance engagement and provide a more dynamic user experience.

Design: Information Structure

Home Page Overview: The home page introduces the organization’s mission and provides an 'About Us' section. Users can navigate to join the movement, learn about ways to contribute, and understand the importance of their involvement. The page concludes with contact information and a footer for easy access to other parts of the site. Testimony Page: This page showcases inspiring stories from those who have benefited from the initiatives, highlighting real-life impacts and emphasizing the value of community support. Donate Page: Here, users can explore donation options. Multiple contribution choices allow donors to select how they wish to fund the organization’s efforts against homelessness. Bible Verses Visual Node: This section presents a collection of inspirational Bible verses aligned with the organization’s mission. Each verse promotes themes of charity and kindness to encourage community support. Homeless Population Graph Visual: This page uses data visualizations to provide insights into homelessness, showcasing trends and statistics that highlight the need for action. Design Page: This page explains the design principles and visual choices of the website. It includes wireframes that align with the organization’s mission and values, enhancing the user experience through thoughtful design elements. Theory Page: This section delves into the theoretical foundations guiding the organization’s initiatives and explains the principles behind its actions. Footer: Each page includes a footer with links and contact information, allowing users to easily access other sections of the site.

Design: User flow

Flow 1 (Read Testimonials): The user reads testimonials by exploring introductory content and tapping "see more" for additional stories. Flow 2 (Make a Donation): The user views donation options, selects an item to donate, proceeds to checkout, and completes the transaction. Flow 3 (Explore Node Visuals): The user reads introductory texts, drags and clicks on nodes for more info, and explores connections. Flow 4 (Interact with Graph Visuals): The user views charts, clicks on them for details, and interacts with buttons for further engagement. Flow 5 (Learn About Design): The user reads about design processes, views wireframes, and scrolls through a slide for deeper insight. Flow 6 (Read Theory Content): The user reads theory sections, explores essays, and navigates between essays using dot navigation.
